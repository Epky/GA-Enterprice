## Introduction

The Staff Product Management System provides comprehensive tools for managing your beauty store's product catalog. This guide will help you understand and effectively use all features available to staff members.

### Key Features

- **Product Management**: Create, edit, and manage products with variants
- **Inventory Tracking**: Monitor stock levels and manage inventory movements
- **Category Organization**: Organize products in hierarchical categories
- **Brand Management**: Manage product brands and manufacturers
- **Image Management**: Upload and organize product images
- **Pricing Control**: Set prices and manage promotions
- **Visibility Management**: Control product visibility and featured status

## Getting Started

### Accessing the System

1. Log in to your staff account at `/login`
2. You'll be automatically redirected to the staff dashboard
3. Navigate to "Products" in the main menu to access product management

### Dashboard Overview

The staff dashboard provides quick access to:
- **Product Statistics**: Total products, active products, low stock alerts
- **Recent Activity**: Latest product updates and inventory changes
- **Quick Actions**: Shortcuts to common tasks
- **Low Stock Alerts**: Products requiring attention

## Product Management

### Creating a New Product

1. **Navigate to Products**
   - Click "Products" in the main menu
   - Click the "Add New Product" button

2. **Fill in Basic Information**
   - **Product Name**: Enter a clear, descriptive name
   - **SKU**: Unique product identifier (auto-generated if left empty)
   - **Category**: Select from the dropdown menu
   - **Brand**: Choose the product brand
   - **Description**: Detailed product description

3. **Set Pricing**
   - **Regular Price**: Base product price
   - **Sale Price**: Optional discounted price
   - **Cost Price**: Your cost (for margin calculations)

4. **Add Product Details**
   - **Weight**: Product weight for shipping
   - **Dimensions**: Length, width, height
   - **Tags**: Keywords for search optimization

5. **Upload Images**
   - Drag and drop images or click to browse
   - Supported formats: JPEG, PNG, WebP
   - Maximum size: 5MB per image
   - Set one image as primary

6. **Configure Variants** (Optional)
   - Click "Add Variant" to create product variations
   - Enter variant name (e.g., "Red", "Large")
   - Set variant-specific SKU and price
   - Manage variant inventory separately

7. **Set Initial Stock**
   - Enter quantity available
   - Set reorder level for low stock alerts
   - Add location information if applicable

8. **Save Product**
   - Click "Save Product" to create
   - Review success message and product details

### Editing Existing Products

1. **Find the Product**
   - Use search bar to find by name or SKU
   - Filter by category, brand, or status
   - Click on product name to view details

2. **Make Changes**
   - Click "Edit" button
   - Update any product information
   - Changes are saved with audit trail

3. **Quick Edit**
   - Use quick edit icons in product list
   - Update price, stock, or status instantly
   - Changes save automatically

### Searching and Filtering

**Search Options:**
- Search by product name
- Search by SKU
- Search by brand name
- Search by category

**Filter Options:**
- Status: Active, Inactive, Out of Stock
- Stock Level: All, Low Stock, Out of Stock
- Featured: Featured products only
- Date Range: Products added in specific period

### Bulk Operations

1. **Select Products**
   - Check boxes next to products
   - Use "Select All" for entire page

2. **Choose Action**
   - Update Status (Active/Inactive)
   - Update Featured Status
   - Export Selected
   - Bulk Price Update

3. **Confirm Changes**
   - Review selected products
   - Confirm bulk operation
   - View results summary

## Inventory Management

### Viewing Inventory Status

1. **Access Inventory Dashboard**
   - Click "Inventory" in main menu
   - View current stock levels
   - See low stock alerts

2. **Inventory Metrics**
   - **Available Stock**: Ready for sale
   - **Reserved Stock**: In pending orders
   - **Total Stock**: Combined quantity

### Updating Stock Levels

**Individual Product Update:**
1. Navigate to product details
2. Click "Update Stock"
3. Enter new quantity
4. Select movement type:
   - Purchase/Restock
   - Sale
   - Adjustment
   - Return
   - Damage/Loss
5. Add notes (optional)
6. Save changes

**Bulk Stock Update:**
1. Go to Inventory > Bulk Update
2. Upload CSV file or enter manually
3. Review changes preview
4. Confirm bulk update
5. View update summary

### Inventory Movements

**Viewing Movement History:**
- Access from product details
- Filter by date range
- Filter by movement type
- Export movement reports

**Movement Information:**
- Date and time
- Movement type
- Quantity changed
- User who made change
- Reference notes
- Before/after quantities

### Low Stock Alerts

**Setting Alert Levels:**
1. Edit product
2. Set "Reorder Level"
3. Set "Reorder Quantity"
4. Save changes

**Viewing Alerts:**
- Dashboard shows alert count
- Inventory page lists all low stock items
- Email notifications (if enabled)

## Category Management

### Creating Categories

1. **Navigate to Categories**
   - Click "Categories" in menu
   - Click "Add Category"

2. **Enter Category Details**
   - **Name**: Category name
   - **Slug**: URL-friendly identifier (auto-generated)
   - **Parent Category**: Select for subcategory
   - **Description**: Category description
   - **Image**: Upload category image

3. **SEO Settings**
   - Meta title
   - Meta description
   - Meta keywords

4. **Save Category**

### Managing Category Hierarchy

**Creating Subcategories:**
- Select parent category when creating
- Unlimited nesting levels supported
- Drag and drop to reorganize

**Reordering Categories:**
1. Go to category list
2. Drag categories to new positions
3. Changes save automatically

### Editing Categories

1. Click category name
2. Update information
3. Save changes
4. Products remain associated

### Deactivating Categories

**Important:** Before deactivating:
- Review products in category
- Reassign products to new category
- Or confirm product handling

## Image Management

### Uploading Images

**Single Upload:**
1. Edit product
2. Click "Add Images"
3. Select file or drag and drop
4. Wait for upload completion
5. Image automatically optimized

**Multiple Upload:**
1. Select multiple files
2. Upload processes all images
3. Progress indicator shows status
4. Review uploaded images

### Managing Product Images

**Setting Primary Image:**
- Click star icon on desired image
- Primary image shows first to customers

**Reordering Images:**
- Drag images to new positions
- Order affects customer gallery display

**Editing Image Details:**
- Click image to edit
- Update alt text for SEO
- Add image caption
- Save changes

**Deleting Images:**
- Click delete icon
- Confirm deletion
- Image removed from storage

### Image Requirements

**Technical Specifications:**
- **Formats**: JPEG, PNG, WebP
- **Maximum Size**: 5MB per image
- **Recommended Dimensions**: 1200x1200px
- **Aspect Ratio**: Square (1:1) preferred

**Best Practices:**
- Use high-quality images
- Ensure good lighting
- Show product from multiple angles
- Include lifestyle shots when possible
- Optimize file size before upload

## Pricing and Promotions

### Setting Product Prices

**Regular Pricing:**
1. Edit product
2. Enter regular price
3. Enter cost price (optional)
4. Save changes

**Variant Pricing:**
- Each variant can have unique price
- Set price adjustment or absolute price
- Variants inherit base price if not set

### Managing Promotions

**Creating Promotions:**
1. Navigate to Promotions
2. Click "Create Promotion"
3. Enter promotion details:
   - Name and description
   - Discount type (percentage/fixed)
   - Discount value
   - Start and end dates
   - Applicable products/categories
4. Save promotion

**Applying Promotions:**
- Select products
- Choose promotion
- Apply to selected
- Promotional price displays automatically

**Scheduling Promotions:**
- Set start date/time
- Set end date/time
- Promotion activates/deactivates automatically

### Bulk Pricing Updates

1. Go to Pricing > Bulk Edit
2. Select products
3. Choose update method:
   - Set fixed price
   - Increase by percentage
   - Decrease by percentage
   - Increase by amount
   - Decrease by amount
4. Preview changes
5. Confirm update

## Best Practices

### Product Creation

✅ **Do:**
- Use clear, descriptive product names
- Write detailed descriptions
- Include all relevant specifications
- Upload high-quality images
- Set accurate stock levels
- Use consistent naming conventions

❌ **Don't:**
- Use vague or generic names
- Skip product descriptions
- Upload low-quality images
- Forget to set reorder levels
- Leave SKUs blank
- Duplicate product entries

### Inventory Management

✅ **Do:**
- Update stock immediately after receiving
- Record all inventory movements
- Set appropriate reorder levels
- Review low stock alerts daily
- Keep movement notes detailed
- Perform regular stock audits

❌ **Don't:**
- Delay stock updates
- Skip movement documentation
- Ignore low stock alerts
- Make adjustments without notes
- Forget to check reserved stock

### Image Management

✅ **Do:**
- Use consistent image dimensions
- Optimize images before upload
- Add descriptive alt text
- Show products from multiple angles
- Set primary image carefully
- Remove poor quality images

❌ **Don't:**
- Upload oversized images
- Use inconsistent backgrounds
- Skip alt text
- Use only one image
- Keep blurry or dark images

### Category Organization

✅ **Do:**
- Create logical category structure
- Use clear category names
- Limit nesting depth (3-4 levels max)
- Review category assignments regularly
- Keep categories organized

❌ **Don't:**
- Create too many categories
- Use confusing category names
- Nest categories too deeply
- Leave products uncategorized
- Duplicate categories

### Pricing Strategy

✅ **Do:**
- Set competitive prices
- Update prices regularly
- Use promotions strategically
- Monitor profit margins
- Test pricing strategies

❌ **Don't:**
- Set prices without research
- Forget to end promotions
- Overlap conflicting promotions
- Price below cost
- Change prices too frequently
